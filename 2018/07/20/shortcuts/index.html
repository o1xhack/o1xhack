Keep 锻炼记录 Day One

v1.5 2018-09-22
@o1xhack

导入问题：

  • 输入 训练营名称，列表
  • 输入 选择有训练营后 DayOne 的标签
  • 输入 选择有训练营后 DayOne 的日记本
  • 输入 选择没有训练营后 DayOne 的标签
  • 输入 选择没有训练营后 DayOne 的日记本

功能：

  • 选择总共锻炼数目
  • 选择从 K3 到 K0 四个等级每个锻炼数目
  • 选择是否有训练营
  • 生成自定义 DayOne，见截图
  • DayOne 中 锻炼数目一行为斜体，训练营名称为粗体

⬇️ DayOne锻炼1.5

有问题请在文章末尾评论，我会及时收到通知！

读书记录 DayOne

v1.1 2018-06-22
@o1xhack

导入问题：

  • 输入 tag
  • 输入 笔记本

功能：

  • 搜索列表结果显示：名称，作者，日期（易于区分）
  • 读取：平均分，作者，日期，豆瓣链接
  • 照片：自选，最后一张，不加入或豆瓣获取
  • 支持星星打分
  • 支持写入书评

⬇️ DayOne书评1.1
